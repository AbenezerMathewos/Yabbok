# YABBOK – Youth Fellowship Management Platform

## Overview

YABBOK is a full-stack web application developed for Youths Strong Fellowship of Kale Hiwot Churches. The platform serves as a centralized system for communication, member management, event updates, and spiritual resources.

## Features

- User registration and authentication
- Member profile management
- Prayer request submission
- Announcements and notifications
- Sermons and audio message management
- Photo gallery
- Books and spiritual resources
- Fellowship meeting schedules
- Live meeting support
- Community chat
- Admin dashboard for managing users and content
- Dark and light theme

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Purpose

The goal of YABBOK is to strengthen communication, fellowship, and spiritual growth among youth from different Kale Hiwot Churches through a modern and user-friendly digital platform.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/yabbok.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
